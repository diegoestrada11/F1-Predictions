Formula 1 Prediction League

This web application built using Node.js, Express.js, EJS, and PostgreSQL. The purpose of the site is to allow users to participate in predicting the results of Formula 1 races and compare their predictions with other users.

Users can create accounts, log in, and submit predictions for upcoming races. Each prediction includes selecting drivers that the user believes will finish in the top positions for a race. After the race is completed, administrators can enter the official race results, and the system calculates points based on how accurate each user’s predictions were.

The application also includes a leaderboard that displays the rankings of users based on their accumulated points throughout the season. This allows users to track their performance and compete with others.

The system includes multiple user roles. Standard users can submit predictions, view race information, and participate in discussions. Moderators can manage user-generated content such as comments. Administrators have full control of the system and can manage races, drivers, teams, and race results.

The website demonstrates several backend development concepts including database relationships, authentication and authorization, server-side rendering, MVC architecture, and secure data handling. All content is stored in a PostgreSQL database and dynamically rendered using EJS templates.

This project is deployed on Render with a connected PostgreSQL database and follows best practices for backend web application development.
